# Phase 21: Set Cargo.toml dev profile debug = 1
**Priority:** P3 — Trivial correctness improvement  
**File:** `Cargo.toml`

## Goal
Restore debug symbols in dev builds so alpha users and contributors get useful backtraces.

## Tasks
1. Change `[profile.dev]` from `debug = false` to `debug = 1`.
2. Keep `[profile.release]` `debug = false` (binary size priority for release).
3. Document in README or a comment in `Cargo.toml` that `RUST_BACKTRACE=1` now produces full traces in dev.

## Validation
- `cargo test` runs with debug symbols.
- `RUST_BACKTRACE=1 cargo test` shows useful line numbers.

## Why Included
Alpha crash reports are useless with `debug=false`. One-line change, high value.
