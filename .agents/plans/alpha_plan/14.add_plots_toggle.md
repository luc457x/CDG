# Phase 14: Add plots toggle (flag + env, default ON)
**Priority:** P1 — High value for alpha testing  
**Module:** `src/pipeline.rs`, `src/main.rs` (base_plan P7)

## Goal
Let alpha users disable PNG generation in constrained environments while keeping it on by default.

## Tasks
1. Add `plots: bool` to `PipelineConfig` (default `true`).
2. In `RunPipeline` CLI (`main.rs:~50`):
   - `#[arg(long, default_value_t = true, env = "CDG_PLOTS")] plots: bool`
   - `#[arg(long = "no-plots")] no_plots: bool`
   - Effective = `plots && !no_plots`.
3. Change gate in `pipeline.rs` from `if !light` to `if config.plots && !light`. Print skip message when disabled.
4. Thread `plots` into `run_standalone_backtest`; gate the equity plot block (`pipeline.rs:~1024-1039`).
5. Add `--no-plots` / `CDG_PLOTS` to the `Backtest` subcommand as well.

## Validation
- `cargo build` clean.
- Manual: `cdg collect --coin btc,eth --no-plots` → no PNG files produced, skip message printed.
- Manual: `cdg backtest --no-plots` → no equity PNG.

## Why Included
Alpha testing often runs in headless CI / low-disk environments. Being able to toggle plots is a small change with high convenience.
