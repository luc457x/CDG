# Load Rules

Rules for loading context files:

- **Onboarding / Init**: Read [structure rules](./structure.md).
- **Every turn**: Read, follow [communication rules](./communication.md) and [safety rules](./safety.md).
- **Coding**: [SPEC.md](../SPEC.md). Read [harness rules](./harness.md), [engineering rules](./engineering.md). Re-read mid-impl.
- **Bug fixes**: [PROGRESS.md](../PROGRESS.md). Read [harness rules](./harness.md), [engineering rules](./engineering.md). Read [PROGRESS_ARCHIVE.md](../PROGRESS_ARCHIVE.md) only if old context needed.
- **Docs**: [workflow rules](./workflow.md).
- **Q&A / Analysis**: Skip reading `.agents/` files unless needed.
